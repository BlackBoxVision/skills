## Adjustment factors

Apply these on top of base rates when relevant:

| Factor | Multiplier |
|--------|-----------|
| Real-time / safety-critical domain | ×1.3–1.5 |
| Regulated industry (fintech, healthtech) | ×1.2–1.4 |
| Rare/niche tech stack | ×1.2–1.3 |
| Startup (equity compensation may offset) | ×0.85–1.0 |
| Legacy stack (COBOL, old Java, ColdFusion) | ×1.3–1.6 |
| Rush delivery | ×1.25–1.5 |

## Agency overhead vs. freelancer vs. internal cost comparison

For a given "billed" rate, the actual talent cost fraction is:

| Team type | Talent cost as % of billing |
|-----------|----------------------------|
| Internal (fully loaded with benefits, office) | 100% (you pay all) |
| Agency (client-facing) | ~45–60% goes to talent |
| Nearshore agency | ~55–70% goes to talent |
| Offshore agency | ~50–65% goes to talent |
| Freelancers | ~85–95% is talent cost |
| AI agent fleet | ~10–30% is talent cost (manager only) |
